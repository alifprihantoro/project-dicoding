    Bagaimana Saya Membuat Website Hanya Bermodal HP

    Hai, Saya Alief Prihantoro. Saat ini saya sedang mendalami typescript, NodeJS, dan React/Next. Dan untuk mencapai saat ini, saya hanya bermodalkan HP saja. Nah bagaimana caranya? Mari kita simak sebagai berikut :

    Persiapan

    Nah untuk persiapannya tidak perlu hal yang mahal. Minimal kalian punya kuota dan HP. Lebih bagus punya keyboard, Mouse dan OTG untuk membantu proses belajar kalian.

    Text Editor

    Untuk text editor sendiri, saya sarankan pakai NVIM. Meskipun susah pada awalnya, namun setelah terbiasa akan menambah produktivitas kalian ketimbang pakai text editor lain. Dikarenakan NVIM sendiri memiliki banyak extension ketimbang yang lain. Bahkan extension di NVIM hampir mirip dengan VS CODE walaupun beberapa plugin tidak ada. Namun jika kalian mau, kalian bisa berkontribusi untuk membuat extension sendiri.

    Saya sudah install NVIM kok tampilannya seperti ini? Nah, agar tampilan kalian seperti gambar diatas silahkan kalian kunjungi repo saya disini. Nah disana Kalian bisa ikuti panduannya, jika ada kendala bisa diskusikan di issues. semoga lancar and happy coding.

    Disarankan untuk menggunakan configurasi termux dibawah

    Configurasi Termux

    Nah bagi kalian yang sering ribet saat commit di github, kalian bisa menggunakan configurasi yang saya buat. Dan kalian bisa ubah beberapa. Karena configurasi termux saya gak terlalu advance, cuman alias dan beberapa function untuk membantu navigasi di command line. Untuk repo-nya sendiri silahkan klick disini.

    Tempat belajar

    Ada banyak website untuk belajar, Namun jika kalian bingung mulai dari mana, bisa pakek cari roadmap.sh Disana udah ada panduannya, walau dalam Bahasa Inggris, namun kalian bisa browser di google dengan keyword tersebut. Dan salah satu sumber informasi paling bagus ialah dokumentasi. Ada juga Youtube untuk mempelajari bassic. Namun jika kalian masih kesusahan bisa ikut ke ecourse atau bootcamp agar ada pembimbingnya.

    Kesimpulan

    Semua hal bisa jadi mungkin asalkan kita berusaha. Tak ada alasan tidak punya laptop/computer. Semoga membantu anda and Happy Coding~
